Desarrollo de sistemas informáticos

Practica: Tareas Iniciales.

Esta primera práctica consiste en la preparación e instalación de una serie de herramientas necesarias para la asignatura:

  • NodeJs. Entorno de ejecución en java con el manipulador de paquetes npm.

  • Git. Sistema de control de versiones.

  • Github. Herramiendo de alojamieto de proyectos online, basado en el control de versiones.

  • Markdown. Metodología de escritura de texto plano con posibilidad de dar formato al texto.

  • C9. Entorno de desarrollo en la nube basado en ubuntu.

  • Gitbook. Herramienta implementada con NodeJs que facilita la construcción de documentación y libros.

  • Pandoc. Herramienta de código abierto que facilita la conversion de documentos entre formatos como Markdown, html o pdf entro otros
